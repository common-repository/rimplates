== Description ==


Rimplates is a dashboard maker for wordpress. Using this Plugin is simple, install it, Rimplates will appear on your admin dashboard menu (with ability to add / create templates, dashboard ~ it supports both admin dashboard & user dashboard). 


> You might need the [Docs - http://rimplates.tawk.help/](http://rimplates.tawk.help/).

== Installation ==
** EASY WAY : 
Search for Rimplates from your WP ADMIN Backend -> Plugins -> Add New, then search for Rimplates, it will show up, then Install & activate.

OR MANUALLY IF YOU HAVE PROBLEM WITH ABOVE

***Download rimplates.zip from https://downloads.wordpress.org/plugin/rimplates.zip
1. Upload `rimplates.zip` via ftp or cpanel upload to the folder `/wp-content/plugins/` directory of your wordpress website installation. OR Login to your wordpress admin dashboard and upload the .zip file
2. Activate the plugin through the 'Plugins' menu in WordPress Admin backend

== Frequently Asked Questions ==

= What are Required to use this Plugin? =

 Install Wordpress on your server, then install this Rimplates Plugin

= How do I Create a New Dashboard? =

Click Rimplates Template on your wordpress admin backend, then click the ADD NEW templates and set the necessary settings, then click on the Publish Button.

= How do I change Dashboard title & other settings =
Visit your Wordpress admin dashboard, click on Rimplates Template-> All Templates, then click Edit on the specific Dashboard you want to edit, then change the neccessary settings and Update, Changes will reflect.   

= What shortcode do I use to display my created dashboard =

Create a new page from wordpress admin backend, and insert the shortcode generated, use shortcode [rimplenet-template id=ID],  e.g if your ID is 5 , your valid shortcode will be [rimplenet-template id=5], for your dashboard to display as whole page, choose Rimplate Blank as template on the page created.
